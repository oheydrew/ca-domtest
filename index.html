Document Object Model!

Howdy folks! Here's a nice little playground for you to mess with. Have a look at the index.html, and take note of the difference between "classes", and "id's".

Remember: Classes are used in stylesheets to set styles. ID's are generally used by javascript to select, and manipulate the element they're assigned to.

Can you:

  • Use document.getElementById to select the spanTwo ID?
  • Use object.innerText to change the text to what it should be?
  • Use object.style to fix the error I made in the third box's backgroundColor?
  • Change the colour of the whole lower container box?

Finally, I've added a .whackybox class to the styles.css - Can you:

  • Use object.className to make the first box whacky?

Have fun!
